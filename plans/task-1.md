# Task 1: Call an LLM from Code - Implementation Plan

## Выбор LLM провайдера
- **Провайдер**: Qwen Code API
- **Модель**: qwen3-coder-plus
- **Причины**: 1000 бесплатных запросов в день, работает из России

## Структура агента
1. Чтение конфигурации из .env.agent.secret
2. Получение вопроса из аргументов командной строки
3. Вызов API с таймаутом 55 секунд
4. Форматирование ответа в JSON с полями answer и tool_calls
5. Вывод только JSON в stdout, логи в stderr

## Обработка ошибок
- Таймаут: 60 секунд максимум
- При ошибках возвращается JSON с описанием ошибки
- Код возврата 0 при успехе, 1 при ошибке
